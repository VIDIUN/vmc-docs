Set Scheduling

Set specific scheduling parameters for your content.

To set a scheduling for an entry

1.  Select the Content tab and then select the Entries tab.

2.  Select an entry or multiple entries and select Bulk Actions on the bottom of the page.

3.  Select Set Scheduling from the drop-down menu.

4.  Enter the scheduling times that should be applied to the selected entries.

5.  Click Apply to all Selected Entries.
