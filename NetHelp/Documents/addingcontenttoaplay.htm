Adding Content to a Player

  To add content to a player

1.   Select the Studio tab.

2.   Highlight the relevant player in the Player List and select Select Content from the Select Action drop down.


The Entries table from the Content tab is displayed.

3.   Select an Entry. You can preview the entry using the player you started off with.

4.   Click Preview and Embed and then grab the embed code and copy it to publish the player to your site.

5.   Click Save and Close.
