AdapTV

To configure companion ads

1.  Add this code to the Additional Parameters and Plugins.

key: adaptv.companionId

value: companiondiv

2.  Add a div with a matching id in the HTML that holds the following player:

<div id="companiondiv" style="width: 300px; height: 250px;">

</div>

Custom SWF
