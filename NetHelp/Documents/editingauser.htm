Editing a User

After a user account is set in the system, the user’s personal details can only be set by the user personally, from the My User Settings page in the Settings tab.  See My User Settings.

To edit user information

1.  Go to the Administration tab and select the Users tab.

2.  Click on a User and then select Edit from the drop down tab.

Only the Publisher User ID and the User Role may be edited in Edit User window. Please note that it is not possible to edit the user role that is defined as the Account Owner. In addition, a VMC user with access to user management functionalities cannot edit the user role selection in their account.
