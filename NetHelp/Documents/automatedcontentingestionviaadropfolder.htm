Using a Drop Folder

The Vidiun Drop Folders are used to automate the ingestion of content into the Vidiun platform. The Drop Folder service constantly watches each drop folder for new content, and activates automatic ingestion of new content to the specific account.

Vidiun offers multiple configuration options for setting each drop folder to a specific workflow. To learn more read about Drop Folders see Vidiun Drop Folder Service for Content Ingestion.
