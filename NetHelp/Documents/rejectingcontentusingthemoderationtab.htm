Rejecting Content Using the Moderation Tab

To reject content (one or more entries)

1.  Select the Content tab and then select the Moderation tab.

2.  Select the relevant entries (for multiple choices, use either the CTRL key or the selection drop-down menu at the bottom of the page) and press the "Reject Selected" button.

You can reject more than one entry at a time, but only for entries appearing on the same page of the main list.
