Duplicating a Player

To duplicate a player

1.  Select the Studio tab and then select Flash Studio.

2.  Click on the relevant player in the Player List.

Players created manually through Vidiun's professional services cannot be edited in this area.

3.  Select Duplicate from the Select Action drop down.


The player configuration Features tab is displayed. See Player Control Options.

4.  Check the control options for the player and then Save Changes.
