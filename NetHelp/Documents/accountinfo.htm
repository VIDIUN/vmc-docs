Account Info

The account info displays account information. The Partner ID is your Vidiun account identification number. The Sub Partner ID is generally deprecated and kept for backward compatibility of older Vidiun based applications.

The Administrator Secret and User Secret are the API private keys used to generate authentication tokens for sessions with the Vidiun servers when using the API. Since the keys can be used to run API commands on your content in Vidiun, you should keep these secret. Usually, the user secret is enough for all activities and therefore this is the key that should be provided to parties wishing to access your Vidiun account via API. The Admin Secret can be used to login as an administrator, and therefore can be used to perform any action on your account.

For more information, please refer to the Vidiun API documentation.

To view additional low-level account settings information

•      Click Advanced Settings to open the advanced Account Settings display. Changing the advanced settings will take effect only for new content ingested (from the moment changes have been saved) and is not retroactive.
