Vidiun Player Supported Plug-ins

The following lists some of the third party plug-ins supported by the Vidiun Player.

•      Google

•      Omniture

•      Nielson

•      Comscore

 

NOTE: When you create a player through the Studio tab in the VMC, the Vidiun Analytics configuration is automatically included.

 

All other plug-ins must be added manually. To learn how to implement a player plugin that sends Analytics data to third-party servers, contact Vidiun.
