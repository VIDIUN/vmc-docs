Different Types of Ad Servers

There are additional ad servers that Vidiun supports, which are a bit more complex than just plugins. For example, Vidiun is now fully integrated with DoubleClick and Freewheel. Their integrations are more complex since there is a server-side connector that allows you to define the metadata to submit to the ad server for ad targeting purposes. The Vidiun metadata gets mapped to ad serverâ€™s metadata. Then, just like with a distribution module, see Vidiun Distribution Module, when a Vidiun partner wants an entry to be included in the ad server, it is marked as such in the VMC. The ad server connector passes all of the metadata to the ad server, where the Vidiun partner can then decide how ads should be targeted based on the metadata.

Vidiun Player Additions Related to Ads

If a Vidiun partner wants to create their own type of ad targeting and prevent pre-rolls from playing on specific entries, they can utilize the flashVar vast.preSequence=0, defined in the embed code in the additional parameters and plugins.

Advanced functionality is enabled to provide for a better workflow of ad plugins. The Vidiun Player build its UI only when the plugin is completely ready, enabling for a smooth loading of player and plugin together after the plugin is ready.
